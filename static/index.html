Microtheory layer (Tier 1). User-asserted facts scoped to a single conversation via session_id. The verifier consults this first — a conversation-specific assertion wins over the cross-session user store.

User memory (Tier 2). Cross-session user assertions (session_id IS NULL). These persist across every conversation and are treated as ground truth.

Model claims. Facts the assistant has asserted in its drafts, plus the verifier's verdict on each. These are NOT consulted by the tiered verifier — they're the system's record of what the model has said and how it landed.

World cache (Tier 3). Retrieval verdicts on world facts, cached for reuse with TTL keyed by stability class.
